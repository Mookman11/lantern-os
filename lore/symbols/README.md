# Symbols

Use this folder for language that is allowed to be meaningful before it is operational.

Each symbol note should use this shape:

```text
Name:
Image:
Felt meaning:
Allowed use:
Blocked use:
Reality check:
Small safe next action:
Review date:
```

Keep private people out by default. Use role labels, fictional placeholders, or redacted descriptions unless there is current consent and a clear reason.
