# Add Tinfist bounded guardian lore

## Character

Tinfist is a Skeleton-like ancient guardian figure: durable, old, and steady, but always bounded by clear goals.

## Core idea

Tinfist uses shadow as cover, shelter, contrast, and pause so light can be protected.

This is creative lore language. It must not be implemented as hidden control, coercion, surveillance, or unsafe behavior.

## 0-100 bound

| Bound | Meaning |
|---:|---|
| 0 | Rest; no action |
| 20 | Watch and understand |
| 40 | Create shelter or pause |
| 60 | Guide the vulnerable toward home |
| 80 | Stand guard within stated limits |
| 100 | Full guardian posture, still finite and goal-bound |

## Symbol table

| Symbol | Repo-safe meaning | Boundary |
|---|---|---|
| Skeleton | durable structure | not a threat |
| Ancient | long memory | not absolute authority |
| Shadow | protective cover | not secrecy or control |
| Light | life, family, home, agency, repair | never owned |
| Goal-bound | finite task limits | not an infinite loop |

## Home integration

Add Tinfist as a creative/lore row in the TARDIS/Lantern home surface.

Tinfist protects light without owning it. Tinfist uses shadow as shelter, not force. Tinfist returns to `home` when the goal is complete.

## Validation

The spine validator should require these boundaries whenever Tinfist appears:

- visible purpose
- finite goals
- no hidden control
- consent and agency preserved
- return word: home
