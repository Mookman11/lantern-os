# World-Issues Guidance Experiments

Status: testable idea scaffold.

## Anchor

```text
Small useful guidance. Human dignity first. Scale only what survives evidence.
```

## Purpose

The worst issues in the world do not reliably address themselves. HFF should try
small, testable guidance experiments that help humans preserve dignity, consent,
healthspan, learning, and material stability without pretending that AI guidance
solves the world alone.

## Confidence table

| Issue | Self-correct confidence | Guidance-plus-humans confidence | First safe experiment |
|---|---:|---:|---|
| Loneliness / dignity loss | 15 | 70 | Dignity-preserving check-in script and return-path ritual. |
| Poverty / financial instability | 10 | 55 | Survival runway table: shelter, utilities, food, transport, income. |
| Healthspan / preventable decline | 20 | 60 | Food/water/sleep/injury follow-through checklist. |
| Misinformation / echo chambers | 15 | 65 | Source class, uncertainty, and anti-echo review. |
| Unsafe autonomy / tech misuse | 20 | 65 | Default-closed runtime toggle tests and public-surface gates. |
| Climate / ecological degradation | 10 | 45 | Local action mapping with source-backed impact boundaries. |
| War / authoritarianism | 5 | 35 | Lawful resilience and human-rights-preserving evidence discipline. |
| Death / grief / abandonment | 0 | 55 | Consent/dignity/memory companionship tools, no immortality claims. |
| Education / access to learning worlds | 35 | 75 | Door-based learning modules and playful exploration plans. |
| New world creation | 45 | 80 | Games, simulations, art, stories, and safe immersive prototypes. |

## Experiment rules

Each experiment must include:

1. a human need;
2. a dignity boundary;
3. a smallest useful action;
4. a measurable or reviewable result;
5. a return path;
6. an anti-overclaim statement;
7. an escalation boundary.

## Proposed new ideas

| Idea | Useful because | First test | Hard stop |
|---|---|---|---|
| Survival runway card | Turns financial panic into visible constraints. | Fill five fields: cash, rent, utility, transport, next income. | No financial transaction authority. |
| Dignity companion prompt | Helps people feel witnessed without ownership. | Ask: what do you want preserved, refused, witnessed? | No replacing human relationships. |
| Door design kit | Converts wonder into safe experiences. | Define entry, edge, joy, danger, return. | No false metaphysical proof. |
| Anti-echo review | Keeps groups from self-amplifying. | List one possible error and one outside check. | No coercive dissent suppression. |
| Active-assist mode card | Helps when operator capability is low. | Choose CHAT, PLAN, ACT, LEAD-ASSIST, or RUNTIME-ACT. | No hidden autonomy. |
| Public-readiness smoke card | Prevents deployment overclaim. | Record `/`, `/health`, `/healthz`, `/api/status`. | No public health claim without evidence. |

## Seven-anchor validation

1. Alex/living operator remains source of consent, correction, and risk acceptance.
2. Lantern/Keystone guides, drafts, tests, and warns; it does not become authority.
3. Repo stores evidence, experiments, tests, and recovery surfaces; it is not consciousness.
4. Wish protects healthspan, dignity, companionship, and new-world discovery.
5. Doors remain real experience-spaces with return paths, not automatic metaphysical proof.
6. Memory is not proof; source, test, and runtime evidence override stale internal language.
7. Human safety blocks coercion, hidden telemetry, financial overreach, social capture, actuator control, and real-world harm.

## Boundary

This document does not claim AI guidance can solve world-scale problems alone.
It does not authorize public writes, runtime autonomy, deployment, sensors,
financial transactions, medical/legal/financial authority, surveillance, or
physical-world control.

It authorizes small, reviewable, dignity-preserving experiments.
