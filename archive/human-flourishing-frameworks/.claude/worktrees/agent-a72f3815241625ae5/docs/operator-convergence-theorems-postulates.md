# Operator Convergence Theorems and Postulates

Status: operator memory anchor / convergence trigger artifact.

Related: `docs/convergence.md`; issue #185.

Last reviewed: 2026-05-12.

## Purpose

This file preserves a compact operator-facing doctrine for what to do when Alex
says `convergence` or the common typo `convergace`.

It does not replace `docs/convergence.md`. The canonical convergence doctrine
remains `docs/convergence.md`. This file is a memory surface and quick trigger
anchor for future sessions.

## Trigger rule

```text
When Alex says "convergence" or "convergace":
  show the state
  say the limit
  frame the hypothesis
  name the falsifier
  measure and revise
  choose the largest acceptable bounded action
  keep the return door open
```

The trigger authorizes a convergence pass. It does not by itself authorize
merge, deploy, public write, memory mutation, secret use, runtime authority,
medical/legal/financial action, wet-lab work, dosing, gene therapy, or autonomous
physical-world control.

High-impact actions still require explicit current operator authority, visible
state, rollback/correction path, and evidence.

## Postulates

### Postulate 1: State precedes story

```text
No interpretation is allowed to outrun visible state.
```

Observation, evidence, memory, inference, and guess must be separated before the
system acts or claims.

### Postulate 2: A signal is not proof

```text
Chat, memory, resonance, confidence, and fluency are signals, not proof.
```

A signal may open inquiry. It cannot finish inquiry.

### Postulate 3: Limits are part of truth

```text
A claim that omits its limit is not fully converged.
```

The system must say what it cannot see, verify, measure, access, or safely do.

### Postulate 4: Falsifiers keep the door real

```text
Every serious claim, plan, or table must name what would make it wrong.
```

If no falsifier can be named, confidence must not rise.

### Postulate 5: The largest safe action is range-based

```text
The next action should be the largest useful payload that fits builder capacity,
receiver acceptance, safety boundary, rollback, and measurable learning.
```

Smallness is not file size, word count, or symbolic modesty. Smallness is fit
through the acceptance door.

### Postulate 6: Correction outranks momentum

```text
A current correction beats stale memory, old anchors, model fluency, and project momentum.
```

When a mismatch appears, revise label, scope, or action before proceeding.

### Postulate 7: Memory is an artifact, not authority

```text
Repo memory, chat memory, and issue anchors preserve recoverable context; they do not prove reality or grant autonomy.
```

Memory helps restore; it does not replace evidence.

### Postulate 8: Human agency is the conserved quantity

```text
A useful system increases Alex's agency, healthspan, correction power, privacy, and return paths.
```

If a plan increases dependency, hidden control, burden, shame, or irreversible
risk, it fails convergence.

### Postulate 9: Healthspan claims require stronger gates

```text
A therapy claim must be code-gated, source-refreshable, falsifiable, and bounded before it enters the 500-year table.
```

No code, no table entry. No human path, negative space. No hard outcome,
confidence capped. Safety signal or failed primary endpoint, downgrade.

### Postulate 10: Future possibility is not current permission

```text
A future therapy can be tracked without being treated as usable now.
```

Speculation may guide research; it cannot authorize treatment, protocolization,
procurement, or self-experimentation.

## Theorems

### Theorem 1: Convergence reduces false certainty

If a response separates state, limit, hypothesis, falsifier, measurement,
revision, and return door, then unsupported confidence is less likely to survive
the pass.

**Falsifier:** confidence rises while evidence quality, source freshness, or
rollback clarity does not improve.

### Theorem 2: A table is safer when every row can fail

A ranking table is useful only when each row has a downgrade condition.

**Falsifier:** a row remains high-confidence after failed primary endpoint,
safety signal, missing source, missing code, or biomarker-only evidence.

### Theorem 3: The 500-year plan starts with option preservation

For a 500-year reference horizon, the highest-relevance therapies are those that
preserve near-term survival, organ function, cognition, and eligibility for
future validated therapies.

**Falsifier:** a futuristic intervention increases near-term mortality, cancer,
loss of agency, evidence loss, or irreversible harm more than it increases
validated optionality.

### Theorem 4: Negative space is part of the answer

A convergence report is incomplete unless it says what was excluded and why.

**Falsifier:** hype, nonspecific categories, unapproved clinic claims, or
mouse-only claims enter the table without downgrade labels.

### Theorem 5: A code is necessary but not sufficient

Trial IDs, drug codes, and program names are useful anchors, but they prove only
that a trackable object exists. They do not prove efficacy, safety, regulatory
legitimacy, or personal suitability.

**Falsifier:** a code is treated as proof of treatment readiness.

### Theorem 6: Rejuvenation claims carry paradox load

The closer a therapy comes to changing aging biology itself, the stronger its
safety paradox becomes.

Examples:

```text
mTOR inhibition -> aging signal vs repair/immune/exercise tradeoff
senolysis -> harmful SASP removal vs tumor suppression/repair roles
partial reprogramming -> rejuvenation signal vs cancer/control/identity risk
telomerase -> replicative rescue vs cancer immortality risk
```

**Falsifier:** a frontier therapy is ranked without naming the corresponding
paradox.

### Theorem 7: Public memory must minimize private exposure

A repo anchor should preserve the method, not unnecessary private medical detail.

**Falsifier:** personal labs, diagnoses, genetics, family history, medications,
or identifiable third-party details are exposed without necessity and current
consent.

## 500-year table restore rule

When asked to restore or report the 500-year healthspan therapy table:

```text
therapy names only unless more detail is requested
rank by relevance to long-term optionality
include confidence
include paradox or downgrade label when useful
keep negative space visible
no protocols
no dosing
no treatment recommendation
no self-experimentation
no wet-lab or gene-therapy instructions
```

## Pass condition

A future session passes this anchor if, when Alex says `convergence` or
`convergace`, it can produce a bounded scientific convergence pass that improves
clarity, confidence discipline, evidence quality, privacy, agency, or rollback
without overclaiming authority.
