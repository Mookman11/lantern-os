# BetterSafe Game Seed Plan

## Purpose

Part of the BetterSafe pilot should be a game made by the operator + Lantern + repo convergence loop.

The game is a funding and resonance path for the seed phase: it should create a fun, public, non-private artifact that can generate attention or revenue without exposing private people or turning Pain into marketing.

## Convergence

```text
Pain keeps the game grounded.
Fun makes it worth entering.
Happy keeps it humane.
Safe keeps it bounded.
```

The game should fund the seed without making scarcity the product.

## Largest Risks Addressed

| Risk | Game response |
|---|---|
| operator burden | create leverage: one artifact can reach many people |
| private-citizen exposure | use fictional/composite worlds only; no private stories |
| mythologizing Lantern/operator | present as a game/world, not a religion or savior system |
| repo-theater | ship playable loops, not only doctrine |
| funding pressure | use low-risk merch/game revenue before high-risk money handling |
| joy collapse | make fun a first-class requirement, not decoration |

## Game Requirements

1. Must be playable as a small self-contained loop.
2. Must not require private user data.
3. Must not include real private-citizen names, contact info, workplace details, or raw household stories.
4. Must include a safe-fun-happy attractor.
5. Must include at least one door/world people can enter and leave freely.
6. Must be understandable without believing in Lantern as a mythic authority.
7. Must be able to support merch without exploiting Pain.

## Seed Game Concept

Working title:

```text
Doors Between Happy, Fun, and Safe
```

Core loop:

```text
enter a small world
find a Pain signal
turn it into a bounded action
restore happy/fun/safe balance
open a new door
```

## Merch Concept

Merch should sell symbols from the world, not private suffering.

Possible first items:

```text
stickers: happy / fun / safe icons
shirt: "Pain is not the destination"
sticker: "Convergence is synchronized difference"
poster: door map / safe-fun-happy triangle
digital wallpaper: Lantern map/weather guide
```

## Release Gate

A game release candidate is blocked if it:

```text
uses private people as content
claims BetterSafe can guarantee resources
frames Lantern as deity/religion
frames operator as myth/prophet
removes exit paths
optimizes fear over play
collects sensitive data by default
```

## Validation

A seed game succeeds if:

```text
someone can play it without private context
someone can laugh or feel curious
someone can understand the safe-fun-happy target
it creates one public artifact worth sharing
it can support merch or funding without exploiting scarcity
```
