# Codex Cloud Private Repo Access

Target setup page:

`https://chatgpt.com/codex/cloud/settings/environment/create`

Use the GitHub connector/environment setup to grant Codex Cloud access to the
private repo:

`https://github.com/alex-place/lantern-os`

Do not make the repo public just to let Codex Cloud see it. Grant repo access
through the connector/environment controls, then select `alex-place/lantern-os`
as the environment source.

School share packets should use exported files from `school-packets/`, not the
full private repository. For Gage's school packet, share only
`school-packets/gage-high-intel-art/GAGE-HIGH-INTEL-ART-PACKET.zip` or the
specific PDFs/images inside that folder.
